# 基于目标的规划

借助Mix Modeler中基于目标的规划功能，您可以根据业务目标（而不只是预算）进行规划。 您可以放心地跨渠道和营销活动高效地分配预算，以最大限度地实现ROI。 有关详细信息，请参阅Mix Modeler文档中的[计划](https://experienceleague.adobe.com/zh-hans/docs/mix-modeler/using/plans/overview){target="_blank"}。
