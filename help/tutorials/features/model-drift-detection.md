# 模型漂移检测

利用Mix Modeler的模型漂移检测功能，可以自动检测模型性能的变化并通知用户。 他们可以立即采取措施，使模型恢复到最佳性能。 有关详细信息，请参阅Mix Modeler文档中的[模型漂移](https://experienceleague.adobe.com/zh-hans/docs/mix-modeler/using/models/insights#model-drift){target="_blank"}。
